CSS Roadmap Checklist

Complete each step of this checklist for CSS Mastery

In progress!

Learn the basics of the web and html

basic stuff like how the web works, what is http(s), basic html tags and ...

In progress!

Learn the basics of CSS

basic CSS concepts like box-model, styling text, different positions & displays

In progress!

Dive deeper to CSS Selectors

take a time to learn CSS Selectors better. learn about selectors Specificity, why styling with ids is bad? what is the difference between pseudo selectors & pseudo elements? ...

In progress!

Learn newer options for creating layouts

learn CSS Flexbox, Grid layout, CSS Columns and their different use cases.

In progress!

Practice What you've learned so far

build a simple website template. go to codepen.io and try to build at least 20 very simple demo projects and compare your solutions with others

In progress!

Learn CSS3 modules

Learn CSS3 modules like Animations/Transitions, Gradients, Transforms, Blend-modes & ...

In progress!

Learn responsive web design techniques

search and practice about different responsive design techniques. media queries, units like em & rem, how to create grid layouts like bootstrap, and different tricks to make rwd easier

In progress!

Pick a CSS Pre-processor

learn one of the many CSS Pre-processors (preferably SCSS). also PostCSS (not technically a pre-processor) is a nice tool to have in your tool chain

In progress!

Go deep! learn about the why!

Search for a deeper understanding of CSS. what is a BFC? what is a Stacking context? why do margins collapse and when do they not?! why animating width/height is a bad idea? but keep in mind that this is a continuous step. start going deeper. but don't get stuck in this step!

In progress!

Take a look at a CSS Framework

briefly examine different CSS Framework options like Bulma, Bootstrap, Materialize. and pick one to use on your next project

In progress!

Learn about CSS Methodologies and Architecture and apply them to your projects

read about different CSS Methodologies like BEM, ABEM, ITCSS, SUITCSS, OOCSS, ... and apply the one you like best

In progress!

Expand your knowledge/experience and get more in touch with the community

Expand your knowledge, follow CSS-related blogs like css-tricks.com or smashing magazine. get in touch with the community, try to find other CSS experts, get familiar with interesting tools & concepts in the community like CSS-in-JS, ...
